## 【初學者】Beginners
#### 【第一次用Linux揀咩Distro好呀？】Which Linux distro should I try first? 

1. [**Nobara**](https://nobaraproject.org/download-nobara/) ：專為Gaming設計`（可以用到最新嘅software）`
2. [**Linux Mint**](https://linuxmint.com/) ：輕量簡單, it works`（Update經時間考驗再推出，所以理應穩定）`
3. [**CachyOS**](https://cachyos.org/) ：最易用嘅Arch`（享用AUR、各種安裝自定義，但唔使煩匘唔識打command）`
> 以上3隻Distro都係跟機有曬Driver(包括Nvidia GPU driver)，唔需要擔心Set up任何嘢可以即時用得。\
> 而且提供Live Environment，咁你只需要一隻USB就可以體驗佢哋，而唔需要安裝落系統。
> ![ScreenshotOfNobara40&MintDesktop&CachyOSInstaller](/Nobara&Mint&CachyOS.jpg)

## 【Journalctl 平常使用方法】

睇journalctl由最尾開始睇  
> `sudo journalctl \-e`

淨係load 最尾嗰1000行  
> `sudo journalctl \-n1000`

淨係睇啱啱開機到現在嘅log  
> `sudo journalctl \-b 0`

睇上一次開機到上一次關機嘅log  
> `sudo journalctl \-b \-1`

show返每句log係由邊一個systemd unit嘔出嚟  
> `sudo journalctl \-o with-unit`

可以用systemd unit嚟filter你想要嘅log, 可以用上面-o with-unit搵返岩嘅systmed unit名  
> `sudo journalctl \-u (systemd unit名)`

filter error log  
> `sudo journalctl \-p3`

Filter kernel log  
> `sudo journalctl \-k`

實時監察  
> `sudo journalctl \-f`

用時間嚟做filter  
> `sudo journalctl \--since` 時間 \`--until` 時間

> 上面 \`--since` \`--until` "時間" 個format用 `man 7 systemd.time` 參考返  
> 其實都超級彈性  
> 就咁打 `sudo journalctl \--since '20min ago' \--until '10min ago'` 咁都得 

> 睇緊log嗰時可以按下 /  
> 之後打字,按enter嚟search  
> 之後按n, search next match  
> 或者按N, search previous match
